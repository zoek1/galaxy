# Loyalty Dapp

## Setup

Clone the repository and enter the Loyalty Dapp directory.

```
git clone https://github.com/zoek1/loyalty-dapp.git
cd loyalty-dapp/dapp
```

## Run

```
yarn start
```

On your browser open [http://localhost:3000](http://localhost:3000)
